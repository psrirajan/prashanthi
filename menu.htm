Setup & Use

Copy files menu.js and null.gif in folder of your page

Write this code in your page:

After this, you have access to Menu object.

Use this sheme (write code at this sheme in your page)

Description Menu object property
Menu.Left // Left coordinate of Menu area (sample: Menu.Left="0px";), default "0px"
Menu.Top // Top coordinate of Menu area (sample: Menu.Top="0px";), default "0px"
Menu.Width // Width of Menu area (sample: Menu.Width="50%";), default "100%"
Menu.Height // Height of Menu area (sample: Menu.Height="150%";), default "100%"

Description CreateMenuBox
Menu.CreateMenuBox(
vis, // Main state visibility of this Menu Box (sample: "show"-anytime static visible, "hide"-dynamic visible)
orientation, // Orientation of this Menu Box (sample: "V"-vertical, "H"-horisontal)
name, // Name of this Menu Box (sample: "Menu01")
left, // Left coordinate of this Menu Box area (sample: "0px")
top, // Top coordinate of this Menu Box area (sample: "0px")
width, // Width of this Menu Box area (sample: "200px")
height, // Height of this Menu Box area (sample: "200px")
maincolor, // Main color of this Menu Box (sample: "#FF00FF")
bordercolor, // Border color of this Menu Box (sample: "#FF00FF")
backimage, // Background image of this Menu Box (sample: "back.jpg")
cellspacing, // Size cell spacing of this Menu Box (sample: "2px")
cellpadding, // Size cell pading of this Menu Box (sample: "2px")
border, // Size border of this Menu Box (sample: "0px")
shadow, // On|Off shadow of this Menu Box (sample: "1"- On shadow, "0"- Off shadow)
shadowcolor, // Shadow color of this Menu Box (sample: "#999999")
classname, // Class name style from CSS of this Menu Box (sample: "menubox")
);

Description MenuBox.Add
menubox.Add( // menubox is a name of MenuBox for this menu item (sample: "Menu01")
name, // Name of this item (sample: "File")
url, // URL for this item (sample: "http://filed.newmail.ru" or "#")
target, // Target for this url item (sample: "newwindow" or "_self")
title, // Title of this item (sample: "This is a title")
mbox, // Name link Menu box of this item (sample: "Menu01" or "")
width, // Width of this item (sample: "100px")
height, // Height of this item (sample: "30px")
align, // Horizontal align of this item (sample: "left" or "center" or "right" or "")
activecolor, // Color of this item, where it activate (sample: "#FF0000")
backcolor, // Color of this item, where it deactivate (sample: "#FFFF00")
backimage, // Background image of this item (sample: "back.jpg")
aclassname, // Class name style from CSS of this item (sample: "menuitem")
);

Description Menu.Run
Menu.Run(); // Run Menu engine
